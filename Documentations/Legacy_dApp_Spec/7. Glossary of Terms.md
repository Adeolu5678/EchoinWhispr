# Part VII: Glossary of Terms

- **Convex:** The backend-as-a-service platform powering the application's data and logic.
- **Clerk:** The authentication provider used for secure user login and identity management.
- **Mutation:** A Convex function that modifies data in the database (write operation).
- **Query:** A Convex function that reads data from the database (read operation).
- **Action:** A Convex function that can perform side effects, such as calling external APIs (e.g., Stripe, OpenAI).
- **Persona:** The user's anonymous profile (Career, Interests, Mood) visible to other users.
- **Storage ID:** A unique identifier for a file stored in Convex File Storage.
- **Subscription:** In Convex terms, a real-time connection that pushes data updates to the client. In App terms, a paid plan for premium features.
