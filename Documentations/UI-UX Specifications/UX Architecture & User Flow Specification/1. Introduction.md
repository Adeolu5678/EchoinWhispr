This document serves as the single source of truth for the EchoinWhispr dApp's user experience (UX), information architecture (IA), and core user flows. It translates the "back-end first" logic (smart contracts, HCS) into a tangible, user-centric design plan.
All visual design, branding, and color are intentionally omitted. This specification focuses purely on structure, hierarchy, and logical flow to ensure a seamless and intuitive experience.
