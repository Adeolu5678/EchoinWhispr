### Flow 1: Project Creation (The Forge)
**Goal:** To take a Founder from idea to published project.
1. **Initiation:** User clicks "Create Project".
2. **Essentials:** Inputs Title, Tagline, Industry. Uploads Logo.
3. **The Pitch:** Inputs Problem, Solution, Market. Uploads Pitch Deck (PDF).
4. **The Ask:** Sets Funding Goal and Equity.
5. **Publish:** System creates Project and Workspace. Status -> "Published".

### Flow 2: Application & Hiring
**Goal:** To connect a Talent with a Project.
1. **Discovery:** Talent finds Project via Search/Dashboard.
2. **Apply:** Talent clicks "Apply", selects Role (e.g., "Co-founder"), writes message.
3. **Review:** Founder receives notification. Views Application.
4. **Interview:** Founder clicks "Start Interview". Chat opens.
5. **Accept:** Founder clicks "Accept". Talent added to Workspace.

### Flow 3: Investment Interest (Soft Circle)
**Goal:** To signal investor interest.
1. **Discovery:** Investor views Project Detail.
2. **Action:** Investor clicks "Soft Circle".
3. **Input:** Enters amount (e.g., $25k).
4. **Confirmation:** System updates "Committed" bar (private/anonymized). Founder notified.
5. **Follow-up:** Investor clicks "Contact Founder" to start Due Diligence chat.
