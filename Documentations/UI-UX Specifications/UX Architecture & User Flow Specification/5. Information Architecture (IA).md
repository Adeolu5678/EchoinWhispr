The dApp is structured around a **Permanent Bottom Navigation Bar** (Glassmorphism).

### ENTRY FLOW
- [Screen: Welcome / Role Selection]
- [Screen: Create Profile]

### MAIN APP (Persistent Bottom Nav)
1. **Dashboard:**
   - [Screen: My Projects] (Entrepreneur)
   - [Screen: Deal Flow] (Investor)
2. **Projects:**
   - [Screen: Project Detail View] (Tabs: Pitch, Team, Updates)
   - [Screen: Create Project Wizard]
3. **Messages:**
   - [Screen: Inbox] (Tabs: Direct, Workspace)
   - [Screen: Chat View]
4. **Profile:**
   - [Screen: User Profile] (Tabs: About, Portfolio, Reputation)
   - [Screen: Settings]
