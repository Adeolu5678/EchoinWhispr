Components are designed to be clean, with clear states and minimal visual noise.
Buttons
Corners: 8px rounded corners on all buttons.
Padding: 12px 24px
Primary Button
Use: The main "call to action" on any screen.
Default: Primary-500 background. White text.
Hover: Primary-600 background. White text.
Disabled: Neutral-700 background. Neutral-600 text.
Secondary (Outline) Button
Use: Secondary actions, e.g., "Manage Subscription."
Default: Transparent background. 1px solid Primary-500 border. Primary-500 text.
Hover: Primary-500 background. White text. (Inverts).
Disabled: Transparent background. 1px solid Neutral-700 border. Neutral-600 text.
Accent Button
Use: Special, "creative" actions, e.g., "Find Mood Match."
Default: Accent-500 background. White text.
Hover: Accent-600 background. White text.
Inputs (Text, Text Area, Dropdown)
Corners: 8px rounded corners.
Default: Transparent background. 1px solid Neutral-700 border. Neutral-100 text.
Focus: 1px solid Primary-500 border. A subtle outer glow: box-shadow: 0 0 0 2px #00A3FF30.
Error: 1px solid Error border. Helper text below input turns to Error color.
Placeholder: Neutral-600 text.
Tags (for Interests)
Default: Pill-shaped (16px border-radius). Neutral-700 background. Neutral-100 text (14px).
Cards / Containers
Default: Neutral-800 background.
Corners: 16px rounded corners.
Border: 1px solid Neutral-700.
Shadow: 0px 4px 12px #00000040 (A subtle shadow to lift it off the Neutral-900 background).
Navigation (Bottom Tab Bar)
Container: Neutral-800 background. 1px top border of Neutral-700.
Icon (Inactive): Neutral-400 color. No text label.
Icon (Active): Primary-500 color. A matching Primary-500 text label (12px) appears below the icon.
Notification Badge: A 5px Accent-500 dot on the icon.
In-Page Tabs (e.g., Inbox)
Container: Transparent. 1px bottom border of Neutral-700.
Tab (Inactive): Neutral-400 text.
Tab (Active): Primary-500 text (Bold). A 2px Primary-500 underline bar.
