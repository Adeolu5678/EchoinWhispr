# Component Styles

Components are designed to be **Premium, Pristine, and Immersive**, drawing inspiration from iOS 16+ design language.

## 1. Global Aesthetic: Glassmorphism
- **Glass Containers:**
  - Background: `rgba(30, 41, 59, 0.7)` (Slate-900 with opacity) or `rgba(255, 255, 255, 0.1)` for light mode.
  - Blur: `backdrop-filter: blur(12px)`.
  - Border: `1px solid rgba(255, 255, 255, 0.1)`.
  - Shadow: `0 8px 32px 0 rgba(0, 0, 0, 0.37)`.
- **Usage:** Navigation bars, cards, modal overlays, and sticky headers.

## 2. Navigation

### Permanent Bottom Menu (iOS Style)
- **Position:** Fixed at the bottom of the screen (z-index: 50).
- **Style:** Floating glass bar with rounded corners (or full width on mobile).
- **Appearance:** Heavy blur (`backdrop-filter: blur(20px)`), translucent background.
- **Items:** Dashboard, Projects, Messages, Profile.
- **Active State:** Icon fills with Primary color, subtle glow.

### Breadcrumbs
- **Requirement:** MUST appear on every page visited from within another page.
- **Location:** Top left of the content area, below the header.
- **Style:** `Home > Parent > Current Page`.
- **Interaction:** Clickable links for parent levels.

## 3. Loading States: Skeletal
- **Requirement:** NO generic spinners for content loading.
- **Behavior:** Use pulsing skeletal shapes that mimic the content layout.
- **Animation:** `animate-pulse` (opacity fade in/out).
- **Color:** `bg-slate-800` (darker) to `bg-slate-700` (lighter).

## 4. Layout Components

### Workspace Member Carousel
- **Display:** Horizontal scrollable list (slider).
- **Card Style:** Glass card containing Avatar + Name + Role.
- **Interaction:** Smooth snap scrolling.

### Buttons
- **Primary:** Gradient background (Indigo-600 to Violet-600), rounded-full (pill shape), subtle shadow.
- **Secondary:** Glass background, white border, white text.

### Cards
- **Style:** Glassmorphism (see above).
- **Corners:** `rounded-2xl` (24px) or `rounded-3xl` (32px) for a modern feel.
