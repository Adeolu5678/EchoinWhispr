# Part VI: External Interfaces and Dependencies

## Hedera Network:

- **Hedera JSON-RPC Relay:** For all EVM smart contract read/write operations.
- **Hedera Mirror Node (gRPC):** For subscribing to the HCS Conversation Topic.
- **Hedera Consensus Service (HCS):** The specific HcsTopicID for conversations.
- **Hedera Token Service (HTS):** The specific HtsTokenID for subscriptions.

## Decentralized Storage:

- **IPFS Gateway (e.g., Infura, Pinata):** An HTTP gateway to upload and download IPFS content.

## Client-Side Libraries:

- **WalletConnect:** For Hedera wallet (HashPack) integration.
- **Hedera SDKs:** hedera-sdk-js.
- **EVM SDKs:** ethers.js.
- **Encryption Library:** ECIES-compatible (e.g., ecies-js).
- **IPFS Library:** ipfs-http-client or equivalent.
- **UUID Library:** For generating client-side convo_id.
