# Part VI: External Interfaces and Dependencies

## Backend & Database

- **Convex:** The core backend-as-a-service platform. Handles database, serverless functions, file storage, and real-time subscriptions.

## Authentication

- **Clerk:** Handles all user authentication, session management, and identity verification.

## Payments

- **Stripe:** Handles subscription payments and checkout flows.

## Frontend Libraries

- **Next.js:** React framework for the application.
- **Convex React SDK:** For connecting to the Convex backend.
- **Clerk React SDK:** For authentication components and hooks.
- **Tailwind CSS:** For styling.
- **Lucide React:** For icons.
