# Part II: System Architecture and Design

## 2.1. Architectural Blueprint (The Modern Web Stack)

Launchpad is a serverless web application leveraging the power of Convex and Next.js to provide a seamless, real-time experience for Entrepreneurs and Investors.

| Layer             | Service / Technology | Role in Launchpad                                                                                   |
| ----------------- | -------------------- | --------------------------------------------------------------------------------------------------- |
| Identity & Auth   | Clerk                | Handles dual-role authentication (Entrepreneur/Investor), session management, and identity verification. |
| State & Logic     | Convex               | Real-time database and backend functions. Manages projects, applications, and deal flows.           |
| Data Storage      | Convex File Storage  | Stores pitch decks, logos, and user avatars securely.                                               |
| Real-time Updates | Convex Subscriptions | Pushes live updates for new applications, messages, and project status changes.                     |
| Frontend          | Next.js (React)      | The responsive user interface, featuring distinct dashboards for different user roles.              |

## 2.2. Data Strategy

This data model is optimized for high-integrity structured data (financials, project specs) and real-time communication.

### Database (Convex):

- **What:** Core platform data.
- **Data:** Users (Entrepreneurs/Investors), Projects, Applications, Deal Discussions.
- **Why:** Convex provides strong consistency and real-time capabilities, ensuring investors always see the latest project data and founders get instant notifications of interest.

### File Storage (Convex):

- **What:** Media and Documents.
- **Data:** Project Logos, Pitch Decks (PDFs), User Avatars.
- **Why:** Integrated storage simplifies the handling of sensitive documents like pitch decks, allowing for strict access control logic within the same backend.

## 2.3. Technology Stack

- **Client (Frontend):** Next.js (React)
- **Authentication:** Clerk
- **Backend & Database:** Convex
- **Styling:** Tailwind CSS (Custom Design System)
- **Payments:** Stripe (Future integration for premium listings/investor tools)
